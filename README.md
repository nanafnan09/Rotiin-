# 🍞 Rotiin - Aplikasi Resep Roti Interaktif

**Rotiin** adalah aplikasi Android berbasis Java yang menyajikan berbagai resep roti lezat lengkap dengan panduan langkah demi langkah. Aplikasi ini tidak hanya menampilkan resep, tetapi juga dilengkapi dengan fitur **Chat AI** terintegrasi untuk membantu pengguna bertanya seputar *baking* dan resep secara *real-time*.

## ✨ Fitur Utama

*   📜 **Katalog Resep Premium:** Menampilkan daftar resep roti (seperti Roti Pisang, Roti Sosis, Roti Abon, dll) dengan desain *Card* 3D yang elegan.
*   🤖 **Asisten Chat AI:** Terintegrasi dengan fitur *smart chat* untuk menjawab pertanyaan seputar resep, takaran bahan, dan tips memanggang.
*   🔍 **Fitur Pencarian:** Kolom pencarian responsif untuk menemukan resep roti favorit dengan cepat.
*   📱 **UI/UX Menarik:** Desain antarmuka yang dirancang khusus (mengacu pada *storyboard* Figma) dengan paduan warna hangat dan elemen desain *custom shape*.

## 🛠️ Tech Stack & Library

Proyek ini dibangun menggunakan teknologi berikut:

*   **Platform:** Android (Minimum SDK 24)
*   **Bahasa Pemrograman:** Java (Native)
*   **Desain UI:** XML dengan Custom Drawable Shapes & Material Design Components
*   **Networking:** OkHttp3 (Untuk integrasi API GROQ Chat AI)
*   **Arsitektur:** Native Android SDK (RecyclerView, CardView, LinearLayout)
*   **Prototyping:** Figma
